How equivalence proposals work
Each equivalence block corresponds to one equivalence proposal. In each proposal, you may include one or more home courses from Técnico and one or more host courses from the destination university.

Use the same block when the listed courses belong to the same equivalence proposal. To propose a different equivalence, click Add equivalence and create a new block.
Warning: if you complete fewer than 30 ECTS at the host university you will not be eligible for an Erasmus scholarship.
